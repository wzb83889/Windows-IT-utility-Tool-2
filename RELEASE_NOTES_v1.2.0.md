# Release Notes - V1.2.0

本版本面向企业设备资产采集、信创产品初筛、批量归集上传和终端交互体验进行了系统性增强。相比上一个发布版本，V1.2.0 不再只是单机命令行检测脚本，而是形成了更完整的“检测、复核、导出、上传、汇总”闭环。

## Highlights

- 新增主菜单级“信创产品判断”，提供评分过程、命中项和综合置信度。
- 完善基础信息、完整硬件信息、单模块检测和系统报告上传的菜单逻辑。
- 新增 WebDAV 批量采集上传方案，支持连通性预检测、成功地址记忆和三格式报告上传。
- 强化 CSV 输出字段，网卡信息与基础检测口径保持一致，并包含启用网卡的 IP 与 MAC。
- 优化主界面、彩色终端、动态进度条、实时检测时间和 Git Bash/MINGW64 兼容体验。
- 生成正式 GitHub `README.md`，便于项目发布和使用说明沉淀。

## Added

### 主菜单与功能入口

- 新增 6 项主菜单结构：
  - 基础硬件信息检测
  - 硬件完整信息检测
  - 信创产品判断
  - 单模块信息检测
  - 系统报告上传
  - 退出程序
- 将“信创产品判断”提升为主菜单独立入口。
- 将原导出功能重组为“系统报告上传”，并保留三个明确入口：
  - 完整检测并导出
  - 批量采集上传提交，面向员工使用
  - 汇总分析共享目录表，面向信息化管理员使用

### 信创产品判断

- 新增信创判断过程展示。
- 按品牌、型号、CPU、操作系统四类规则计算综合置信度。
- 输出品牌命中、型号命中、CPU 命中、系统命中等复核字段。
- 增强信创判断运行过程进度展示。

### 批量采集与 WebDAV 上传

- 新增 WebDAV 归集上传逻辑。
- 默认 WebDAV 地址：

```text
http://192.168.2.252:19000/DeviceReports
```

- 默认账号：

```text
debug
```

- 密码运行时输入，不做本地持久化保存。
- 新增 WebDAV 连通性测试流程：
  - OPTIONS 探测服务
  - 创建测试目录
  - 上传测试文件
  - 清理测试文件
  - 访问测试完成
- 连通性失败时提前终止，不再等待完整硬件扫描后才提示上传失败。
- 新增最近一次成功归集地址记忆功能。
- 支持更换 WebDAV 地址，并保留失败后的重新输入逻辑。

### 批量采集登记字段

- 批量采集时新增员工登记流程：
  - 选择或输入部门
  - 输入设备使用人员姓名
  - 输入设备使用位置，也就是门牌号
- 部门选择支持预设部门，也支持直接输入其他部门名称。
- 报告归档结构调整为：

```text
归集根目录/
└─ 部门/
   └─ 门牌号/
      ├─ 部门-姓名-计算机名-报告日期.txt
      ├─ 部门-姓名-计算机名-报告日期.json
      └─ 部门-姓名-计算机名-报告日期.csv
```

### 报告与汇总

- 支持同时生成 `txt`、`json`、`csv` 三类报告。
- 新增共享目录 CSV 汇总能力，便于信息化管理员形成设备信息统计表。
- CSV 字段按简易资产登记口径输出。
- CSV 网卡信息与基础硬件信息检测口径保持一致，仅输出启用网卡，并保留每块网卡的 IP 和 MAC。

## Changed

### 检测信息结构

- 基础硬件信息检测保留简洁结果，不显示来源信息。
- 完整硬件信息检测继续保留全量模块信息。
- 硬盘驱动器信息调整为物理硬盘驱动器和逻辑磁盘两类汇总。
- 去除不稳定或无效的磁盘序列号、接口速率、健康度字段。
- 显卡、音频、电池、显示器等输出排版进一步整理。
- 音频设备检测去除驱动日期字段，减少无效字段和卡顿风险。

### 主板、系统、内存信息增强

- 主板和 BIOS 信息新增：
  - 主板名称
  - 主板芯片组
  - DMI 系统版本
  - DMI 主机版本
- 操作系统信息新增：
  - Windows 版本归属发行代号
  - 产品激活状态
- 内存信息新增：
  - DDR 代数识别
  - 外形规格类型，例如 SODIMM
  - 内存插槽数量及占用情况

### 网络与笔记本网卡复核

- 新增笔记本网卡核实提示：
  - 笔记本仅检测到 WiFi 设备时提示核实。
  - 有线网卡禁用时提示核实。
  - 可能未集成有线网卡设备时提示核实。
- CSV 网卡字段不再输出大量无关适配器、WAN Miniport、调试适配器等项目。

### 终端交互与视觉

- 主界面标题更新为：

```text
Windows 企业IT信创系统信息采集工具（数智与信息化发展部 V1.2.0）
```

- 主界面功能说明调整为“功能亮点”。
- 主界面功能亮点图标和文字严格对齐。
- 标题包裹条根据标题宽度动态生成并居中显示。
- 主菜单检测时间支持实时分秒刷新。
- 增加 ANSI/VT 彩色输出支持。
- 兼容 Windows Terminal、PowerShell、Git Bash / MINGW64。
- 色彩规则调整为：
  - 主标题和标题框：梅红色
  - 菜单主体：白色
  - 注释说明：灰色
  - 运行中进程：蓝色
  - 正常、完成、命中：绿色，前缀 `✓`
  - 需核实、警告：黄色，前缀 `⚠`
  - 错误：红色，前缀 `✗`
- 动态进度条改为完整方块填充，百分比显示在进度条前方。

## Fixed

- 修复部分环境下异常字符串打印失败导致闪退的问题。
- 修复音频设备检测耗时过长的问题。
- 修复批量归集在局域网外环境下完整扫描后才提示共享路径失败的问题。
- 修复 CSV 网卡字段输出大量无关适配器的问题。
- 修复本地测试地址可能影响下一次 WebDAV 默认地址的问题。
- 修复动态进度条在较窄终端下被截断的问题。
- 修复功能亮点使用 emoji 时在不同终端中宽度不一致导致的排版漂移问题。

## Build

项目目标框架：

```text
.NET Framework 4.8
```

解决方案：

```text
SystemInfoDebug.sln
```

MSBuild 示例：

```powershell
msbuild .\SystemInfoDebug.sln `
  /t:Build `
  /p:Configuration=Debug `
  /p:Platform="Any CPU" `
  /v:minimal
```

## Notes

- 信创产品判断为规则化辅助判断，建议结合采购台账、设备铭牌和人工复核形成最终结论。
- WebDAV 密码仅运行时输入，不写入本地记忆文件。
- 批量采集前建议确认 WebDAV 地址、端口、账号权限和网络连通性。
- 某些硬件字段是否可读取取决于设备厂商、驱动、BIOS/DMI 信息完整度和当前系统权限。
