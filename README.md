# 系统信息检测工具（数智与信息化发展部 V2026.0.1.0）

## 简介

本工具专为 Windows 系统设计，可快速检测计算机的硬件配置、操作系统信息、品牌型号、序列号、内存 DDR 代数与频率、设备类型（台式机/笔记本）、出厂日期参考值以及信创产品判断。支持基础配置与完整配置两种模式，信息逐行输出，并可一键保存为桌面报告文件。

## 功能亮点

- **双模式检测**：基础配置（系统、品牌、CPU、内存、磁盘、显卡） / 完整配置（额外包含主板、BIOS、网卡、声卡、电池、显示器等）
- **品牌中文识别**：自动将制造商英文名转换为中文（如 LENOVO → 联想）
- **设备类型判定**：根据 `PCSystemType` 自动识别台式机、笔记本或服务器
- **内存详细信息**：显示每根内存条的容量、DDR 代数（DDR3/DDR4/DDR5）及运行频率
- **出厂日期参考**：基于 BIOS 发布日期给出参考出厂日期，并与当前年份对比，偏差过大时提示手动到官网核实
- **信创产品判断**：根据品牌是否国产及 CPU 是否为国产架构（兆芯、海光、飞腾、龙芯等）自动判定
- **报告导出**：按 Enter 键将完整信息保存到桌面，文件名包含计算机名和时间戳
- **交互式菜单**：支持主菜单选择、信息逐行显示、Esc 返回/退出

## 运行环境

- Windows 7 / 8 / 10 / 11（64 位推荐）
- .NET Framework 4.8 或 .NET 6/8 运行时（开发时需引用 `System.Management`）

## 使用方法

1. 下载或编译生成 `SystemInfo.exe`
2. 双击运行，进入主菜单
3. 选择 `1`（基础配置）或 `2`（完整配置）
4. 等待信息采集完成，结果将逐行显示在控制台
5. 显示完毕后：
   - 按 **Enter** 键：将报告保存到桌面（`系统检测报告_计算机名_时间戳.txt`）
   - 按 **Esc** 键：返回主菜单
6. 在主菜单按 `0` 退出程序

## 编译说明

### 使用 Visual Studio

1. 创建 **控制台应用** 项目（.NET Framework 4.8 或 .NET 6/8）
2. 添加 `System.Management` 引用：
   - .NET Framework：右键项目 → 添加 → 引用 → 勾选 `System.Management`
   - .NET Core/5+：通过 NuGet 安装 `System.Management`
3. 将 `Program.cs` 替换为本仓库提供的代码
4. 生成解决方案（`Ctrl+Shift+B`）
5. 运行 `bin\Debug` 或 `bin\Release` 下的可执行文件

### 直接使用预编译包

若已提供 `SystemInfo.exe`，可直接双击运行，无需任何额外依赖。

## 注意事项

- 出厂日期基于 BIOS 发布日期，并非绝对精确，仅供参考。若偏差过大，程序会给出官方售后链接供手动核实。
- 信创产品判断依赖品牌和 CPU 关键词库，若发现误判可自行修改 `domesticBrands` 和 `domesticCpuKeywords` 数组。
- 部分旧机型可能无法获取内存 `MemoryType` 或 `Speed` 值，此时会显示“未知类型”。
