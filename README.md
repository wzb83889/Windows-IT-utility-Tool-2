# 🖥️ Windows 企业IT信创系统信息采集工具 V1.1

> 一款面向团队资产管理的 Windows 硬件信息检测工具，支持模块化检测、批量归集、信创产品智能评分。

## 📌 简介

本工具专为 **Windows 系统** 设计，可快速检测计算机的硬件配置、操作系统信息、品牌型号、序列号、内存 DDR 代数、硬盘接口类型、网卡状态等。支持**单模块调试**、**批量采集归集**、**多格式报告导出**，并内置信创产品智能评分，极大提升 IT 设备盘点效率。

---

## ✨ 功能亮点

| 类别 | 主要功能 |
|------|----------|
| **检测模式** | 基础配置（系统、品牌、CPU、内存、磁盘、显卡）<br>完整配置（额外包含主板、BIOS、网卡、声卡、电池、显示器等） |
| **模块化运行** | 支持单独检测操作系统、内存、硬盘、网卡、打印机、信创判定等 13 个模块 |
| **批量归集** | 命令行一键采集，自动按部门/人员/位置归档到共享目录，生成 txt/json/csv 三格式报告 |
| **统计汇总** | 扫描归集目录，生成设备信息统计表 CSV，便于资产台账管理 |
| **品牌中文识别** | 自动将制造商英文名转换为中文（如 `LENOVO → 联想`） |
| **设备类型判定** | 根据 PCSystemType 自动识别台式机、笔记本或服务器 |
| **内存深度检测** | 显示每根内存条的容量、DDR 代数（DDR3/DDR4/DDR5）、运行频率、插槽位置、部件号 |
| **硬盘增强识别** | 识别接口类型（NVMe/SATA/USB）、品牌、健康状态及 SSD 寿命提示 |
| **网卡状态提示** | 区分无线/有线网卡，笔记本场景下提供连接核实建议 |
| **外围设备检测** | 支持已安装打印机的品牌、型号、驱动版本、序列号、IP 地址、连接方式 |
| **出厂日期参考** | 基于 BIOS 发布日期给出参考出厂日期，并与当前年份对比，偏差过大时提示手动核实 |
| **信创产品判定** | 采用**评分制**（品牌 30 分、CPU 35 分、操作系统 20 分、型号 15 分），输出置信度与命中关键词 |
| **报告导出** | 支持 txt（可读文本）、json（结构化数据）、csv（汇总表格）三种格式，文件名自动包含计算机名、部门、人员等信息 |
| **交互式菜单** | 彩色控制台、动态进度条、单模块调试、命令行无人值守模式 |

## ⚙️ 系统要求

- **操作系统**：Windows 10 / Windows 11（64 位推荐）
- **运行时**：.NET Framework 4.8 或 .NET 6/8（如果使用独立发布则无需安装）
- **权限**：读取 WMI 信息（普通用户即可），批量采集需共享目录写入权限

---

2️⃣ 命令行批量采集（IT 管理员适用）
# 采集本机信息并上传到默认共享目录
SystemInfo.exe --collect

# 指定共享目录
SystemInfo.exe --collect "\\server\设备信息采集文件夹"

# 归集统计已有采集数据
SystemInfo.exe --merge "\\server\设备信息采集文件夹"

批量采集时需输入部门、使用人员、使用位置，一次录入即可自动归档。

💡 常见问题
Q: 程序提示“无法访问共享目录”？
A: 请确认网络路径可写，并具有相应权限。可先用 net use 测试。

Q: 信创判定结果不准怎么办？
A: 工具基于关键词和评分规则，可根据实际需要调整 domesticBrands、domesticCpuKeywords 等数组。

Q: 能否在 Windows 7 上运行？
A: 支持，但部分 WMI 属性可能缺失，建议使用 Windows 10/11。

Q: 如何更新内置的部门列表？
A: 修改 Program.cs 中的 DepartmentOptions 数组，重新编译即可。
